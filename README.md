# GitHub Final Project - Simple Interest Calculator

This project is a simple tool designed to calculate the interest earned or paid over a specific period.

## Formula
The calculation is based on the standard formula:
$A = P(1 + rt)$

Where:
* **P** = Principal amount
* **r** = Annual interest rate (decimal)
* **t** = Time (years)

## Features
* Simple command-line interface.
* Fast and accurate interest calculations.
* Easy-to-read output.
